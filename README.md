## 📺 What Can You Do?

| Feature | Description |
|---|---|
| 🔍 Search | Search any anime — Naruto, One Piece, 2023 anime |
| 🎭 Genre Filter | Action, Romance, Horror, Isekai and 20+ genres |
| 📅 Year Filter | Filter any year from 1960 to 2024 |
| 📡 Airing Now | See what anime is currently airing live |
| 🆕 Upcoming | Browse upcoming anime releases |
| 📖 Full Details | Synopsis, IMDb rating, episodes, studio info |
| 📺 Where to Watch | Netflix, Crunchyroll, Prime Video links |
| 🤖 AI Recommendations | Smart similar anime suggestions |
| ⭐ Favourites | Save your favourite anime |

---

## 🧠 How It Works
```
You search "Naruto" or "2023 anime" or "romance"
              ↓
GNC Anime calls MyAnimeList API (Jikan)
              ↓
20,000+ anime fetched in real time
              ↓
Results shown with ratings, genres, synopsis
              ↓
Click any anime → Full details + Where to Watch + Recommendations
```

---

## 📡 Data Source
Powered by **[Jikan API v4](https://docs.api.jikan.moe/)**
- ✅ 20,000+ anime titles
- ✅ Live real-time data
- ✅ Covers 1960 to 2024
- ✅ No API key needed
